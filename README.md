# 収益物件 購入判断シミュレーター

物件価格と家賃を入れるだけで、手残り・リスク耐性・デッドクロス・売却出口まで1画面で試算できる不動産投資の購入判断ツールです。

**▶ 使ってみる： https://hiroshi5566.github.io/realestate-simulator/**

- 総合判定（必須条件の足切り＋100点スコア・全基準を開示）
- 税引前の月次手残り／税引後の30年キャッシュフロー・IRR
- デッドクロスの発生タイミングを可視化
- エリアの将来人口（全1,903市区町村・社人研データ）を判定に反映
- 保存・比較・PDFレポート出力

## 免責

本ツールは簡易試算です。税金は概算であり、実際の投資判断は必ず詳細な資金計画・現地確認・専門家への相談のうえで行ってください。計算結果について作者は一切の責任を負いません。

人口データ出典：国立社会保障・人口問題研究所「日本の地域別将来推計人口（令和5（2023）年推計）」
